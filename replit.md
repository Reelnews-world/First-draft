# First Draft — HTML/CSS/JS Starter App

## Overview
A simple static starter app using plain HTML, CSS, and JavaScript, served by a lightweight Node.js HTTP server.

## Project Structure
- `index.html` — Main HTML page
- `style.css` — Stylesheet
- `app.js` — Client-side JavaScript
- `server.js` — Node.js static file server (serves on port 5000)

## Running the App
The app is served by `server.js` using Node's built-in `http` module.
- Host: `0.0.0.0`
- Port: `5000`

## Deployment
Configured as a **static** deployment with the root directory (`.`) as the public directory.
